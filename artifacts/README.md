# Artifacts directory

Place Phase 1 Colab outputs here:

| File | Required for |
|------|----------------|
| `model_v2.keras` | Live demo (primary) |
| `model_v1.keras` | Ablation / evaluate |
| `model_v3.keras` | Ablation / evaluate |
| `label2idx.json` | All inference |

Optional mirror for Cursor sprint imports (same files):

```powershell
# Windows PowerShell (from repo root)
New-Item -ItemType Directory -Force -Path output\artifacts, output\src
Copy-Item artifacts\* output\artifacts\ -Recurse -Force
```

```bash
# Linux / macOS / Git Bash
mkdir -p output/artifacts
cp -r artifacts/* output/artifacts/
```

Evaluation arrays (`X_mlp_val.npy`, `y_val.npy`, …) are generated by:

```bash
python scripts/export_eval_arrays.py
```

Requires `data/landmarks/` and `data/augmented_landmarks/` from the Phase 1 notebook.
